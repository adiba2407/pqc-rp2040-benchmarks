# FIPS 203 (ML-KEM)

This folder contains the implementation and benchmarking results for the NIST FIPS 203 Module-Lattice-Based Key Encapsulation Mechanism (ML-KEM) on the Raspberry Pi Pico (RP2040).

## Hardware Platform
- Raspberry Pi Pico (RP2040)
- ARM Cortex-M0+
- 133 MHz CPU
- 264 KB SRAM
- 2 MB Flash

## Software
- Raspberry Pi Pico SDK
- ARM GNU Embedded Toolchain
- PQClean Reference Implementation

## Operations Benchmarked
- Key Generation
- Encapsulation
- Decapsulation

## Results
The benchmark results are provided in the `results` folder.

## Source Code
The implementation files are provided in the `source` folder.
