# FIPS 204 (ML-DSA)

This folder contains the implementation and benchmarking results for the NIST FIPS 204 Module-Lattice-Based Digital Signature Algorithm (ML-DSA) on the Raspberry Pi Pico (RP2040).

## Hardware Platform
- Raspberry Pi Pico (RP2040)
- ARM Cortex-M0+
- 133 MHz CPU

## Software
- Raspberry Pi Pico SDK
- ARM GNU Embedded Toolchain
- PQClean Reference Implementation

## Operations Benchmarked
- Key Generation
- Signature Generation
- Signature Verification

## Results
Benchmark results are located in the `results` folder.

## Source Code
Implementation files are located in the `source` folder.
