# FIPS 205 (SLH-DSA)

This folder contains the implementation and benchmarking results for the NIST FIPS 205 Stateless Hash-Based Digital Signature Algorithm (SLH-DSA) on the Raspberry Pi Pico (RP2040).

## Hardware Platform

- Raspberry Pi Pico (RP2040)
- ARM Cortex-M0+
- 133 MHz CPU
- 264 KB SRAM
- 2 MB Flash

## Software

- Raspberry Pi Pico SDK
- ARM GNU Embedded Toolchain
- CMake
- SLH-DSA Reference Implementation

## Operations Benchmarked

- Key Generation
- Signature Generation
- Signature Verification

## Results

Benchmark results are located in the `results` folder.

## Source Code

Implementation files are located in the `source` folder.
